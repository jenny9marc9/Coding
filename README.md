# Programmierbeispiele

Dieses Code-Portfolio enthält 3 Aufgaben.

### 1. R_Skript: Interaktives Tic-Tac-Toe ('tictactoe.R')
* Funktionen um gegen den Computer oder eine weitere Person Tic-Tac-Toe zu spiele.
  
### 2. Python-Skript: Konvexe optimierung ('optimisationconvexe.py')
* Iterative optimierung einer Funktion über die Methode des negativen Gradienten.

### 3. Python-Skript: Planetenbahn ('Astrodyn.py' & 'IERS_14_C04_IAU2000_act.txt')
* Auswertung und Darstellung von Daten einer Planetbahn
